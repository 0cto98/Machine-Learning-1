A Simple Regression TaskÂ¶

In this first exploration we will consider the regression task in which we are given a training set S={(x1,y1),…,(xm,ym)}S={(x1,y1),…,(xm,ym)} of examples (xi,yi)(xi,yi), where:

  • xi∈Xxi∈X is called the input or the instance,
  • XX is the domain set,
  • yi∈Yyi∈Y is called the output, the target or the label and
  • YY the label set.

Our strategy to solve the regression task is to learn a model h:X→Yh:X→Y or to teach a learner to predict the ouputs associated to the inputs.

A Simple Regression ModelÂ¶

As a simple learner we will consider a nearest neighbor regressor. The nearest neighbor prediction rule is based on pooling the outputs of the nearest instances to take a decision. Given an instance z∈Xz∈X the predicted output of a k−k−nearest neighbor regressor will be

h(z)=1k∑i∈Nk(z,S)yih(z)=1k∑i∈Nk(z,S)yi
where Nk(z,S)Nk(z,S) is the set of the kk nearest instances in SS to zz according to a pre-specified distance.

How good is the prediction?Â¶

To measure how good the prediction rule hS(x)=y^hS(x)=y^ learned over training set SS is for an example (x,y)(x,y) we compute the squared residual error (y^−y)2(y^−y)2. To measure how good the prediction is on a set of examples we compute a loss function for example the mean squared error (MSE). The training error is the loss function computed over the training set:

l(S,hS)=1m∑(xi,yi)∈S(hS(xi)−yi)2l(S,hS)=1m∑(xi,yi)∈S(hS(xi)−yi)2

The training examples where generated by an unknown distribution DD. Another measure of success is how good is our prediction rule on unseen examples generated by the same distribution DD. The generalisation error is the loss function computed on a test set EE sampled from distribution DD:

l(E,hS)=1mE∑(xi,yi)∈E(hS(xi)−yi)2l(E,hS)=1mE∑(xi,yi)∈E(hS(xi)−yi)2

Question 3Â¶

Let Y=(y1,…,ym)Y=(y1,…,ym) and Y^=(y^1,…,y^m)Y^=(y^1,…,y^m). The explained variance between YY and Y^Y^ is:

explained.var(Y,Y^)=1−σ2Y−Y^σ2Yexplained.var(Y,Y^)=1−σY−Y^2σY2
Where the variance of the sample YY can be computed as:
σ2Y=(1m∑i=1my2i)−Y¯2,σY2=(1m∑i=1myi2)−Y¯2,
and the variance of the gap between the targets and the prediction:
σ2Y−Y^=(1m∑i=1m(yi−y^i)2)−(Y−Y^¯¯¯¯¯¯¯¯¯¯¯¯¯¯)2=MSE(Y,Y^)−(Y¯−Y^¯)2.σY−Y^2=(1m∑i=1m(yi−y^i)2)−(Y−Y^¯)2=MSE(Y,Y^)−(Y¯−Y^¯)2.
The explained variance can be rewrote as:
explained.var(Y,Y^)=1−MSE(Y,Y^)−(Y¯−Y^¯)2(1m∑mi=1y2i)−Y¯2explained.var(Y,Y^)=1−MSE(Y,Y^)−(Y¯−Y^¯)2(1m∑i=1myi2)−Y¯2

The explained variance is a performance measure (the higher the better).

1−explained.var(Y,Y^)1−explained.var(Y,Y^) is a renormalised version of MSE(Y,Y^)MSE(Y,Y^).
