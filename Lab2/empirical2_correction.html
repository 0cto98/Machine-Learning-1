Some learning tasks call for nonlinear predictors, such as polynomial predictors. Take, for instance, a one dimensional polynomial function of degree nn, that is, p(x)=a0+a1x+a2x2+…+anxn.p(x)=a0+a1x+a2x2+…+anxn. One way to learn this class is by reduction to the problem of linear regression. To translate a polynomial regression problem to a linear regression problem, we define the mapping ψ:R→Rn+1ψ:R→Rn+1 such that ψ(x)=(1,x,x2,…,xn)ψ(x)=(1,x,x2,…,xn). Then we have that p(ψ(x))=a0+a1x+a2x2+…+anxn=a⋅ψ(x)p(ψ(x))=a0+a1x+a2x2+…+anxn=a⋅ψ(x) and we can find the optimal vector of coefficients aa by using the Least Squares algorithm.

from sklearn.metrics import explained_variance_score
#from unknown_distrib import data_sample

def poly_reg_emp_gen_errors(train, test, d_vals):
    """
    For each d value in d_vals this function computes the empirical and 
    generalization errors on the train and test sets of the polynomial 
    regression model of degree d found by least square minimization.
    
    :param train: a 2 columns (x,y) pandas dataframe, the training set
    :param test: a 2 columns (x,y) pandas dataframe, the test set 
                 generated by the same distribution as the training set
    :param d_vals: a list of intergers, the degrees of the polynomial regressors 
    :Returns a pandas dataframe with the errors associated to the degree
    """
    X_train = np.vstack(train.x)
    y_train = train.y

    X_test = np.vstack(test["x"].values) # column matrix instead of line vector
    y_test = test["y"]

    gen_errors = []
    emp_errors = []
    for d in d_vals:
        # Train
        poly_reg_lin = make_pipeline(PolynomialFeatures(d),
                               LinearRegression())
        poly_reg_lin.fit(X_train, y_train)
        # Predict
        yhat_train = poly_reg_lin.predict(X_train)
        emp_errors.append(1-explained_variance_score(y_train, yhat_train))#mean_squared_error(y, yhat_train))
        yhat_test = poly_reg_lin.predict(X_test)
        gen_errors.append(1-explained_variance_score(y_test, yhat_test))#mean_squared_error(y_test, yhat_test))
        
    d_errors = pd.DataFrame({"d":d_vals,"empirical":emp_errors,"generalization":gen_errors})
    return d_errors
