Questions:¶

  1. What is the domain set of the Iris dataset?
  2. Consider only 2 dimensions: visualise the dataset and the classes
  3. Recode the classes to get a binary classification problem :
    • Iris setosa : -1
    • Iris Versicolor & Iris Virginica : 1
  4. Use the perceptron algorithm, implemented in the scikit-learn library, to separate the 2 classes. Compute the empirical and generalization errors. Visualize the decision function (use the weights $w$).
  5. Use the logistic regression algorithm to separate the 2 classes. Compute the empirical and generalization errors. Visualize the decision function (use the weights $w$).
  6. Recode the classes
    • Iris Virginica : -1
    • Iris Setosa & Iris Versicolor : 1
  7. Repeat the questions 4 and 5 for this classification problem

Answers:¶

Question 1¶

The domain set ${\cal X}$ of Iris dataset is a subset of $\mathbb{R}^4$.

Question 3 : Recode the classes to get a binary classification problem (separable case)¶

In [6]:
ynew = y.replace([0,1,2],[-1,1,1])

Xtrain = X[["sepal width","petal length"]].sample(frac=0.5)
ytrain = ynew.loc[Xtrain.index] # The associated output
# test set = whole set \ train set
Xtest = X[["sepal width","petal length"]].loc[X.index.difference(Xtrain.index)] 
ytest = ynew.loc[Xtest.index] # The associated output
